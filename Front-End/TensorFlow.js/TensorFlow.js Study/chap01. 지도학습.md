### how to do Supervised learning?
1. 과거의 데이터 준비 (과거의 데이터를 통해 원인과 결과의 관계를 파악)
    - 원인 : 독립변수
    - 결과 : 종속변수
2. 모델의 모양 만들기
3. 데이터로 모델을 학습 (FIT)
4. 모델 이용
