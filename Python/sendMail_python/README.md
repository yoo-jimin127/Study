### SMTP
- simple mail transfer protocol
- EMAIL SERVER to EMAIL SERVER : SMTP
- EMAIL SERVER to EMAIL CLIENT : IMAP
- EMAIL CLIENT to EMAIL SERVER : SMTP
- SMTP 서버를 이용해 우리가 원하는 곳으로 메일 보낼 수 있음
